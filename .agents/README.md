# Generic Agent Context

This directory contains concise, tool-agnostic guidance for coding agents working on this project.

## Structure

- `project-context.md` — architecture, domain, and important implementation notes.
- `skills/posthog/` — current PostHog integration guide and event inventory.
- `workflows/` — short task-specific checklists.

## Relationship to `.claude/`

`.claude/` contains the original Claude Code skill bundle generated by a PostHog wizard. It includes long reference docs that are still useful, but they are verbose and Claude-specific.

For everyday work, start with:

1. `AGENTS.md`
2. `.agents/project-context.md`
3. Any relevant file under `.agents/skills/` or `.agents/workflows/`

Only use `.claude/skills/posthog-integration-expo/references/` when deeper PostHog SDK documentation is needed.
